STUDENT COURSE REGISTRATION SYSTEM

Project Description

The student course registration system is a command-line python project that helps training school manage students, course and course registrations.

===Core functions:===
Add new students
View all students
Search for students
Add new courses
View all courses
Register students for courses
View students registered in a course
View courses registered by a student
Save data to JSON files
Load saved data from JSON file


==Tools:===
mainly python -> OOP, inheritance, file handling , error handling , functions, dictionaries and lists
command-line interface


====How to run the assignment:===
Open terminal 
Navigate to the project folder
Run the application (python3 main.py)

===classes used====
Person -> base class that stores name, email and phone number

Student -> inhereits from the person class and adds student id

Course -> represents course and stores course id , course name , trainer name and capacity

SchoolSystem -> manages students, courses and registration and saving and loading data


===Features=====+
student management
course management
registration management
data persistence 
validation and error handling
